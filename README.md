# Netson Solar System 

**Netson Solar System** is a full-stack web platform developed to showcase solar energy services, completed installation projects, and promote renewable energy solutions.
The platform also spreads awareness about government solar initiatives such as the PM Surya Ghar Muft Bijli Yojana, encouraging households to adopt rooftop solar systems and reduce electricity costs.

---

##  Features

* Solar company portfolio and service information
* Showcase of completed solar installation projects
* Information about rooftop solar solutions
* Awareness about government solar subsidy schemes
* Clean and responsive user interface
* Full-stack architecture with frontend and backend modules

##  Installation & Setup

npm start

Run backend server

cd backend
npm install
npm run dev

 About PM Surya Ghar Muft Bijli Yojana

PM Surya Ghar Muft Bijli Yojana is a Government of India initiative that promotes rooftop solar installation for households across the country.
The scheme provides financial subsidies to encourage families to generate their own solar power and reduce electricity bills.

**Key Benefits**

* Subsidy support for rooftop solar systems
* Lower electricity costs for households
* Promotion of clean and renewable energy
* Energy independence for homes
